# Pre-Commit

### 1. Open nano edit

```bash
nano .git/hooks/pre-commit
```

### 2. Remove All text in file ```Ctrl + K```
### 3. Copy new text from ```docs/pre-commit/pre-commit.sh```
- Copy ```Ctrl + Shift + V```
- Save ```Ctrl + O```
- Aprove ```Enter```
- Exit ```CTRL + X```


### 4. Check permissions
```bash
chmod +x .git/hooks/pre-commit
```
